Workbook Build Up!

This section showcases the progressive development of my workbook, combining interactive elements, design sketches, and final outcomes.

HomePage Sketch

For the homepage, I wanted to make my design more interactive (especially since this class is literally called Interactive Media). I pushed myself beyond my comfort zone and used JavaScript to create an interactive element. Through trial and error, I adjusted the parameters for the suitable text moving distance, setting a minimum distance of 20 VH/VW.

Engaging Animation

Next, I thought about how to create a personal identity and style to make the website more engaging. I incorporated hand-drawn animated characters as elements on my page, each featuring a vivid color.

Interactive Elements

I used multiple :hover effects with transform: scale and transition: ease to create smooth size-changing animations for the elements, making the website more interactive.

Challenges and Improvements

Image Linking

One challenge that I faced was to correctly link the images. The image worked perfectly on live Server but when uploaded to the Github, the image was not displayed. After several trials, I was able to fix the issue by renaming the file name: don't leave any spaces in the file name, use dash to subsitute the space.

Items Outside Box

Another difficulty that I faced was to adjust the correct CSS. Sometimes, the items were outside the box . I was able to fix the issue by adjusting the CSS properties: margin, padding, and width, display options and flexbox properties to make sure the items are wrapped inside the box
